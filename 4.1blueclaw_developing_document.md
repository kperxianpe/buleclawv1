### 4.2 核心模块

#### 4.2.1 ThinkingEngine
**文件**: `core/thinking_engine.py`

```python
class ThinkingEngine:
    def analyze(user_input: str) -> ThinkingResult
    def _recognize_intent(text: str) -> Tuple[IntentType, float]
    def _generate_thinking_steps(intent, input) -> List[ThinkingStep]
    def _generate_options(intent, input) -> List[ThinkingOption]
    def execute_option(result, option_id) -> Dict
```

**算法**:
- 意图识别: 关键词匹配 + 正则模式匹配
- 置信度: 加权得分归一化
- 选项生成: 意图模板映射

#### 4.2.2 ExecutionBlueprintSystem
**文件**: `blueclaw/core/execution_blueprint.py`

```python
class ExecutionBlueprintSystem:
    def load_blueprint(steps_data) -> List[ExecutionStep]
    def execute_blueprint() -> ExecutionResult
    def pause_execution()
    def resume_execution()
    def replan_from_step(step_index, new_blueprint)
```

**状态机**:
```
PENDING -> RUNNING -> COMPLETED
            ↓
         PAUSED -> RUNNING
            ↓
         FAILED -> REPLAN
```

#### 4.2.3 SkillRegistry
**文件**: `blueclaw/skills/skill_registry.py`

- 技能注册与发现
- 权限管理 (PermissionLevel: Low, Medium, High)
- 技能执行调度

#### 4.2.4 MemoryManager
**文件**: `blueclaw/memory/memory_manager.py`

- **WorkingMemory**: 当前任务上下文
- **LongTermMemory**: 历史任务记录
- 语义搜索与检索

### 4.4 API 接口

```python
# 创建协调器
coord = create_coordinator_v3(use_real_execution=True)

await coord.start_task("用户输入")

# 处理用户响应
await coord.handle_user_response(response, response_type)

await coord.handle_intervention(action)
```
