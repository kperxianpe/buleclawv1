## GUI 界面

### 启动方式

```bash
# Windows
START_GUI.bat

# Or directly
python blueclaw_v1_gui.py
```

### 界面布局

```
┌─────────────────┬─────────────────────┬────────────────┐
│                 │                     │                │
│   💬 对话        │    📋 执行蓝图       │     📜 日志    │
│   (30%)         │    (50%)            │    (20%)       │
│                 │                     │                │
│  用户输入       │  ⏳ 步骤1: 查询天气   │ [10:00:01]    │
│  AI回复         │  🔄 步骤2: 搜索景点   │ 任务开始...    │
│                 │  ⏳ 步骤3: 规划路线   │               │
│  [发送]         │                     │ [暂停][继续]   │
│                 │                     │                │
└─────────────────┴─────────────────────┴────────────────┘
```

### 功能

- **左侧**: 对话面板，支持快速任务按钮
- **中间**: Canvas 执行蓝图，实时显示步骤状态
- **右侧**: 日志面板，暂停/继续/重新规划按钮
- **弹窗**: 选项选择对话框、问题回答对话框

## 使用说明

1. **启动 GUI**
   ```bash
   python blueclaw_v1_gui.py
   ```

2. **输入任务**
   - 在左侧输入框输入任务
   - 或点击快速任务按钮

3. **查看执行**
   - 中间面板显示执行蓝图
   - 步骤状态实时更新

4. **干预操作**
   - 点击暂停按钮暂停执行
   - 点击重新规划回退到思考阶段
